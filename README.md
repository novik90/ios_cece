# ce·ce

> one ball at a time.

iOS-приложение для ведения счёта в снукере.

## Требования

- iOS 17.0+
- Xcode 15+

## Запуск

1. Клонировать репозиторий
2. Открыть `cece.xcodeproj` в Xcode
3. Выбрать симулятор или устройство
4. `Cmd + R`

## Стек

- SwiftUI
- SwiftData
- MVVM

В разработке
